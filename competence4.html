Projet Automatisation Docker

En équipe de 4 nous avons du réaliser la génération du documentation technique de différent programme ainsi qu'un markdown tout cela dans un conterneur, à l'aide de script bash et php en transformant une page HTML en documentation pdf.

Etape 1

La premère étape du projet consiste à réaliser deux modèles pour les différentes pages webs. Une pour la documentation des programmes puis une pour la markdown. On devait aussi préparer 3 différents programmes en utilisant une méthode pour le commenter différente de d'habitude.

Etape 2

La deuxième étape est donc la création des scripts dockers en bash pour pouvoir générer les pages en pdf. Il fallait au même moment fusionner du php avec le html réaliser précédement afin qu'il puisse lire et réecrire les commentaires et catégories. Puis à partir du conteneur docker il fallait automatiser la génération de la documentation en html/php et en markdown en documentation pdf
